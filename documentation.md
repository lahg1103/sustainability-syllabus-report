# Sustainability Syllabus Report
### How to run
make sure you have python installed, run this command on terminal.

```
python --version
```
additionally, make sure git is installed.

```
git version
```
I will be using bash exclusively for any terminal work. I strongly recommend using bash to follow along.
Make sure to change to the correct directory you want to copy this project to.


<details>
<summary> Windows </summary>
I'm using bash for my terminal, so a WSL has to be installed or use gitbash.

```bash
git clone https://github.com/lahg1103/sustainability-syllabus-report.git
cd sustainability-syllabus-report.git
python -m venv venv
source venv/Scripts/activate
pip install -r requirements.txt
python app.py
```
</details>
<details>
<summary>MacOS</summary>
If you're on MacOS, you already have bash installed. Proceed on terminal.

```bash
git clone https://github.com/lahg1103/sustainability-syllabus-report.git
cd sustainability-syllabus-report
python -m venv venv
source venv/bin/activate
pip install -r requirements.txt
python app.py
```
  
</details>

### Contributing to the Repository
Always, make sure you pull from the working branch, and make sure that your virtual environment is active. Make sure your requirements are up to date.

```bash
pip install -r requirements.txt
```
